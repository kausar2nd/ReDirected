Welcome to the Simple URL Redirect Service

This project helps you set up your own lightweight URL redirect system!

How does it work?

Enter a keyword after the domain name, like this: yourdomain.com/github.
If a keyword is registered, you'll be automatically redirected to the associated link.

Want to create your own?

Fork the repository and follow the simple steps to set up your own redirect service.
